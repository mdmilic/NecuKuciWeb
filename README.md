# NecuKuciWeb
Web application that will support and enhance NecuKuci mobile applications

## Setup Development Machine
1. Pull the code to dev machine and `cd <project_root>`
2. Open client
    ```
    cd client
    npm run dev
    ```
3. Open server
    ```
    cd server
    npm start
    ```
4. Open `http://localhost:8080` in browser
